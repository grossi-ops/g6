JAVASCRIPT FUNCTIONALITY:
===================================================================

showLevel(lvl, btn)
  - Shows/hides prompt blocks based on selected level
  - Removes 'active' class from all blocks and buttons
  - Adds 'active' class to selected block and button
  - Used by all 7 level tabs in prompt panel

copyPrompt(id)
  - Copies prompt text to clipboard
  - Changes button text to "Copied!" for 2 seconds
  - Restores original button text after timeout
  - Works with all 7 prompt levels
